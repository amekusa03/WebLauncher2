# あわ茶 (WebLauncher2)

あわ茶は、特定のWebサイトをまるで独立したアプリのように起動・管理できるAndroidアプリです。
らん茶（5スロット・1階層）の後継として、階層構造によるグループ管理を導入しています。

## 特徴

- **階層構造**: 最大4階層のサブ画面により、「会社名」や「目的」でURLを整理できます。各階層に5枠まで登録可能。
- **専念モード**: URLを開くと同時にアプリが自動終了し、ブラウザから戻る操作が不要になります。
- **カラーアイコン**: プリセット16色 × 名前の先頭文字で識別しやすい円形アイコンを採用。
- **ホーム画面ショートカット**: 任意のURLやサブ画面へ直接ジャンプするショートカットをホーム画面に追加できます。

## 仕組み

1. 管理画面でスロットにURL、またはサブ画面へのリンクを登録する。
2. アイコンをタップすると、URLの場合はブラウザで開き（アプリは自動終了）、サブ画面リンクの場合は次の階層を表示する。
3. 「ホーム画面へ追加」から、特定のURLやサブ画面へのショートカットをホーム画面に作成できる。

## 操作

| 操作 | 動作 |
| ------ | ------ |
| 登録済みアイコンをタップ | URLを開く、またはサブ画面へ進む |
| 未登録枠をタップ | 登録画面を開く |
| アイコンを長押し | 登録・編集画面を開く |
| 「←」ボタン | 1階層上に戻る |
| パンくずのアイコンをタップ | その階層へ直接ジャンプ |

## プロジェクト構成

| 項目 | 技術 |
| ------ | ------ |
| データ管理 | Room Persistence Library（parentIdによる階層構造） |
| 画面遷移 | Jetpack Compose Navigation |
| ブラウザ | Chrome Custom Tabs |
| ショートカット | ShortcutManager API |
