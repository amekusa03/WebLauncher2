# あわ茶(WebLauncher)

あわ茶は、Androidのホーム画面に特定のWebサイトを開くためのショートカット（スロット）を最大16つまで追加できるアプリです。
らん茶の５つでは不足であったユーザーが対象です。。

## 特徴

- **マルチスロット構成**: 4x4、最大16のURLを登録できるランチャーアイコンを利用可能。
- **お湯呑みデザイン**: 各スロットとアプリ本体に、色分けされた円柱型（お湯呑み）アイコンを採用。
- **簡単カラーカスタマイズ**: 16進数を知らなくても、カラーパネルから直感的にアイコン色を選べます。

## プロジェクト構成

### 画面（Activity）

1. **設定・管理画面 (`MainActivity`)**:
    - アプリの設定や各スロットの有効化・無効化、URLの割り当てを行うためのメインUIです。
2. **ブラウザ起動ロジック**:
    - Chrome Custom Tabs を利用し、ホーム画面に作成した各ショートカットから直接Webサイトを開きます。
    - `LauncherActivity` は不要になり、ShortcutManager から Intent を発行する構成へ移行します。

### スロット (`activity-alias`)

※ WebLauncher2 では `activity-alias` による静的な16スロット制限を廃止し、動的にホーム画面へ追加する方式に変更されました。


## 仕組み

1. 管理画面でスロットにURLを設定し保存する。
2. アプリをタップすると、第一段階のURLまたは第２段階を表示するアイコンが表示される、
　　第一段階のURLまたは第２段階が表示されるかの条件は、事前にユーザーが第２段階のアイコンを登録したときである。
　　 各階層には１〜４つアイコンが登録可能。
　　すなわち、フォルダーとファイル構成に類似しており、それぞれの階層には４つまでの登録可能としている
3. 「ホーム画面に追加」機能により、ユーザー指定のラベル名でショートカットを作成可能。
